
# Playwright Automation Testing Framework

##  Project Overview
This project is a complete **end-to-end test automation framework** built using **Playwright + TypeScript**.

It covers:
-  UI automation testing
-  API testing
-  Authentication using storageState
-  CI/CD integration with GitHub Actions

The framework is designed with **real-world best practices** and simulates how automation is built in production environments.

---

## Tech Stack

-  Playwright (UI Automation)
-  TypeScript
-  Node.js
-  GitHub Actions (CI/CD)
